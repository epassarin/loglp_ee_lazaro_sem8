# Atividades Práticas - 2º Ano Ensino Médio

Este repositório contém as atividades de programação desenvolvidas durante o segundo ano do Ensino Médio na **EE Lázaro**, focadas no aprendizado de lógica de programação e na linguagem Python.

## 📌 Sobre o Projeto
As atividades aqui presentes fazem parte do currículo técnico/prático, visando o desenvolvimento de habilidades em:
* Estruturas de decisão (`if`, `else`, `elif`).
* Operadores lógicos (`and`, `or`, `not`).
* Resolução de problemas através de algoritmos.
* Sintaxe da linguagem Python.

## 🚀 Tecnologias Utilizadas
* **Python 3**
* **Ambiente:** VS Code / Google Colab / Laboratório de Informática
